## 为什么保留两版

**v2 的价值**：完整覆盖 100 年所有十个纪元，是世界观骨架。

**v3 的价值**：每一章用文学化叙事深度展开，前 7 纪（2026-2095）已成稿，文笔接近正式出版水平。

**为什么不合并**：v2 与 v3 的文风、视角、时间颗粒度差异较大，硬合并会破坏 v3 的文学统一性。建议未来某天 v3 补完后 8/9/10 三纪，再考虑废弃 v2。
