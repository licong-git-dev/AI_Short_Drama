## 风格定位

每章 2,500-4,000 字，沉浸式叙事，包含：
- 完整的时间、地点、人物状态
- 感官细节（气味、温度、光线、声音）
- 至少一段实质对话（10 个来回以上）
- 内心独白或意识流段落
- 历史事件须从角色视角体验，避免全知旁白

详细规格见 [`正式版_深度创作/00_总览与创作说明.md`](正式版_深度创作/00_总览与创作说明.md)。

## 主要人物（核心档案）

- **陈薇**：1997 年生于上海，软件工程师，全卷见证者（2026=29 → 2126=129 岁）
- **田中浩 / Hiroshi-1**：1953 年生，神经科学教授，2076 年上传成意识体
- **田中健**：1971 年生，Hiroshi 之子，2113 年 142 岁去世
- **Kaito Tanaka**：1973 年生，Hiroshi 之孙，全卷与意识体祖父对话
- **林悦**：2100 年生，基因优化人类，全卷成长主线
- **林明**：林悦之父，火星外交官（与 09 章 Maya 父亲"陈明"无关，注意区分）
- **Prometheus**：2086 年初始化的飞船意识体，在虚空中飞行 38 年
- **Cora Vasquez**：2094 年火星出生，第一代"火星父母的火星孩子"
