# 推广营销（第16-24周）

## 一、推广三阶段

```
预热期（上线前2周）→ 爆发期（上线第1-2周）→ 长尾期（上线后持续）
```

## 二、预热期（上线前2周）

### 2.1 短视频引流（核心）
- **切片投放**：从成片中截取高能片段
  - EP1：苏雪晴悬崖转身 + "你来晚了23年"
  - EP20：悬崖坠落瞬间
  - EP80：苏雪晴说"我爱你"化为光
  - EP95：改写时间线重逢
- **投放平台**：抖音/快手/视频号/B站
- **数量**：每天2-3条切片，持续2周
- **关键**：每条切片结尾"想看完整版？搜索《星海23年》"

### 2.2 话题预埋
- **抖音话题**：
  - #星海23年
  - #等你23年值不值
  - #8401天的等待
  - #她苍老了他却依然年轻
- **小红书话题**：
  - "让我哭最惨的短剧"
  - "科幻版虐恋天花板"
  - "追了100集的短剧推荐"

### 2.3 KOL合作
| 类型 | 数量 | 预算 | 说明 |
|------|------|------|------|
| 短剧推荐号 | 5-10个 | 500-2000/条 | "今天推荐一部科幻虐恋短剧" |
| 情感博主 | 3-5个 | 1000-5000/条 | "你愿意等一个人23年吗" |
| 影视解说 | 2-3个 | 2000-8000/条 | 详细解说前5集 |

## 三、爆发期（上线第1-2周）

### 3.1 信息流投放（付费推广）
- **平台**：巨量引擎（抖音）/ 磁力引擎（快手）
- **投放素材**：5-10条不同切片
- **投放策略**：
  - 先小额测试（每条100-500元）
  - 找到ROI最高的素材
  - 加大投放（日预算3000-10000元）
- **目标**：充值ROI > 1.5（投1元赚1.5元）
- **定向人群**：18-45岁女性，短剧爱好者，科幻/虐恋标签

### 3.2 投放素材类型
| 类型 | 示例 | 转化率 |
|------|------|--------|
| 悬念型 | "她等了23年，他终于来了——但她已经老了" | 高 |
| 情感型 | "看到第80集我哭崩了" | 高 |
| 反转型 | "以为是科幻片，结果虐哭我" | 中高 |
| 对比型 | "她满头白发，他还像离开时一样年轻" | 中 |

### 3.3 互动运营
- 评论区人设：以"编剧"身份回复高赞评论
- 设置互动话题："如果你是林远航，会先去找女儿还是爱人？"
- 每集发布后跟进观众反馈，实时调整

## 四、长尾期（持续运营）

### 4.1 二创激励
- 鼓励观众二创剪辑
- 发起"#星海23年二创大赛"
- 提供高清素材包给UP主/博主

### 4.2 跨平台分发
- 抖音独家期结束后 → 快手/微信/红果同步上线
- 小说版 → 番茄/七猫持续更新
- B站 → 完整版/解说版

### 4.3 话题持续发酵
- 每10集做一次"剧情回顾+解读"
- EP50（林小萤被杀）专门做话题引爆
- EP80（苏雪晴牺牲）做"泪点排行榜"
- EP100（结局）做"彩蛋解读"

## 五、推广预算参考
