## 一、对接策略总图

```
A 类：短剧制作公司（直接拍摄）  → 卖剧本（30-80 万）或联合出品
B 类：短剧分发平台（投流上线）  → 卖完整成片或分账模式
C 类：短剧 MCN/工作室            → 联合开发或 IP 衍生
D 类：长视频平台（横屏改编）    → IP 授权（50-300 万）
E 类：编剧服务平台（挂牌零售）  → 散卖大纲/样本/单集
F 类：经纪/版权代理              → 全权代理出售
```

**首轮投递建议**（W3-W4）：
- **A 类 5 家** + **B 类 2 家** + **E 类 2 家** = 共 9 个并行投递
- 不要把所有底牌一次性给所有人；A 类签 NDA 后才发完整 sample_pack
- 收集反馈，按"询单率/出价范围"做第二轮聚焦

## 二、A 类｜短剧制作公司（首轮主力）

### A1. 听花岛（《我在八零年代当后妈》出品方）
- **背景**：现象级爆款短剧《八零年代当后妈》制作方，擅长年代+情感
- **官方渠道**：通过其母公司"麦芽传媒"官方公众号「麦芽传媒MAIYA」联系商务
- **匹配度**：⭐⭐⭐⭐ — 情感虐恋类有制作经验，《星海23年》情感内核契合
- **投递话术建议**：强调"虐恋核心 + 视觉反差（白发 vs 年轻）"，淡化科幻元素

### A2. 点众科技
- **背景**：百亿级短剧公司，旗下有"河马剧场"、"点众短剧"
- **官方渠道**：官网 dianzhongkj.com → 商务合作
- **匹配度**：⭐⭐⭐⭐⭐ — 题材广，对差异化短剧有承接能力
- **投递话术**：突出 100 集完整剧本 + 可拍性 8.0/10 + 五大爆点

### A3. 完美世界（短剧业务线）
- **背景**：上市公司，短剧业务覆盖国内+海外（ReelShort 体系合作）
- **官方渠道**：官网 → 投资合作 → 短剧合作邮箱
- **匹配度**：⭐⭐⭐⭐ — 出海能力强，《星海23年》情感内核普世
- **投递话术**：附 [`10_business_plan.md`](10_business_plan.md) 中"出海机会窗口"章节

### A4. 九州文化（短剧子品牌）
- **背景**：以"虐恋"题材见长，《重生之我在八零追老婆》等
- **官方渠道**：抖音搜「九州短剧」官方号 → 私信合作
- **匹配度**：⭐⭐⭐⭐ — 虐恋赛道头部
- **投递话术**：强调 EP-78 化光段是"全场泪崩"级催泪

### A5. 容量短剧（小程序剧）
- **背景**：腾讯系，专注小程序剧分发
- **官方渠道**：官方公众号「容量短剧」→ 商务合作
- **匹配度**：⭐⭐⭐ — 体量适中，议价空间合理
- **投递话术**：突出"投递包内含 4-5 集精选样本"，降低对方评估成本

## 三、B 类｜短剧分发平台（首轮 2 家）

### B1. 红果短剧（字节系）
- **背景**：免费短剧平台，月活 2.12 亿（快速增长中）
- **官方渠道**：app 内"创作者中心" → 编剧入驻；或邮箱 hongguo-bd@bytedance.com（公开商务邮箱）
- **匹配度**：⭐⭐⭐⭐⭐ — 红果是当下最热渠道，覆盖三类核心受众都强
- **投递话术**：参考 [`30_市场调研_选题策略/03_付费转化与平台策略.md`](../30_市场调研_选题策略/03_付费转化与平台策略.md)，强调"现实主义+情感深度"

### B2. 抖音剧场（短剧版块）
- **背景**：抖音官方短剧入口
- **官方渠道**：抖音搜「抖音剧场」官方号 → 商务合作 / 通过经纪公司接入
- **匹配度**：⭐⭐⭐⭐ — 流量大，但付费墙模式偏弱
- **投递话术**：强调话题点丰富（每 10 集 3+ 话题），适合二创裂变

## 七、F 类｜经纪/版权代理（W6+ 可选）

### F1. 中文在线（IP 全权代理）
- **背景**：上市公司，IP 运营全链路
- **官方渠道**：官网 → 商务合作
- **匹配度**：⭐⭐⭐ — 适合"全部交给代理人，自己专注创作"的模式
