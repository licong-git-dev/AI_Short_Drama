# 制作筹备（第1-4周）

## 一、两条路径选择

### 路径A：自己制作
- **优势**：利润全归自己，完全控制质量
- **劣势**：需要资金和团队，风险自担
- **适合**：有制作经验或资金实力

### 路径B：卖剧本/合作制作
- **优势**：零成本，快速变现
- **劣势**：利润分成，可能被改得面目全非
- **适合**：纯编剧身份，希望低风险

### 路径C：找投资方
- **优势**：借别人的钱拍自己的戏
- **劣势**：需要商业计划书，要让出部分控制权
- **适合**：有项目但缺资金

## 二、路径A：自己制作的筹备清单

### 2.1 核心团队（最少配置）
| 角色 | 职责 | 参考日薪 | 工期 |
|------|------|---------|------|
| 导演 | 现场调度+镜头语言 | 2000-5000 | 全程 |
| 摄影师 | 拍摄+灯光 | 1500-3000 | 拍摄期 |
| 剪辑师 | 后期剪辑+字幕 | 200-500/集 | 后期 |
| 化妆师 | 造型+特效妆 | 800-1500 | 拍摄期 |
| 场务 | 场地+道具+协调 | 500-800 | 全程 |

> 极简团队：导演兼摄影 + 剪辑 + 化妆 = 3人即可开拍

### 2.2 选角要点
- **林远航**：30-40岁男演员，硬朗型，少话但有爆发力
- **苏雪晴**：需两个演员或特效化妆
  - 年轻版（27岁闪回）：25-30岁女演员
  - 苍老版（50岁主线）：40-50岁女演员 或 年轻演员+老年特效妆
- **选角渠道**：
  - 短剧演员经纪：抖音/快手搜索"短剧演员"
  - 横店群演转型：成本低，有镜头经验
  - 影视院校学生：成本最低，形象好
  - 平台推荐：部分短剧平台提供演员库

### 2.3 场景规划
| 场景 | 频次 | 解决方案 |
|------|------|---------|
| 希望星基地（室内） | 高频 | 工业风仓库改造 / 影棚搭景 |
| 悬崖边 | 多次 | 实景拍摄（山区） / 绿幕合成 |
| 飞船内部 | 高频 | 搭建简易控制台 + 灯光氛围 |
| 太空/星球外景 | 中频 | 绿幕 + 后期特效 |
| 洞穴 | 第21-30集 | 实景溶洞 or 搭景 |

> 科幻题材关键：灯光 > 布景 > 特效。好的灯光能让简单场景看起来很高级

### 2.4 特效需求评估
| 特效类型 | 集数 | 难度 | 方案 |
|---------|------|------|------|
| 飞船降落 | EP1等 | 中 | 3D模型+合成 |
| 星球外景 | 多集 | 低 | Midjourney生成背景+绿幕合成 |
| 苏雪晴化为光 | EP80 | 高 | 粒子特效 |
| 高维空间 | EP81-95 | 高 | 概念化处理，光效为主 |
| 苍老化妆 | 全程 | 中 | 专业老年妆 |

> AI辅助方案：用Midjourney/Stable Diffusion生成概念图和背景，大幅降低特效成本

### 2.5 预算表（中等制作）

| 项目 | 费用(万) | 说明 |
|------|---------|------|
| 演员 | 15-25 | 主演3人+配角+群演 |
| 团队 | 10-15 | 导演+摄影+剪辑+化妆+场务 |
| 场地 | 5-10 | 影棚+实景+交通住宿 |
| 设备 | 3-5 | 摄影机+灯光+收音 |
| 特效 | 5-15 | 科幻场景+粒子效果 |
| 后期 | 5-8 | 剪辑+调色+音效+配乐 |
| 其他 | 2-5 | 餐饮+保险+应急 |
| **合计** | **45-83万** | |

## 三、路径B：卖剧本的操作

### 3.1 投递渠道
- **短剧平台收稿**：
  - 抖音短剧征稿（搜索"抖音短剧征稿"获取最新邮箱）
  - 快手星芒短剧
  - 腾讯微短剧
  - 咪蒙旗下/银色大地等头部制作方
- **编剧平台**：
  - 云莱坞（剧本交易平台）
  - 橘子编剧（短剧对接）
  - 编剧帮（行业社区+项目对接）
- **社交渠道**：
  - 微信公众号/小红书搜索"短剧征稿"
  - 豆瓣编剧小组
  - 影视行业微信群

### 3.2 投递材料准备
- **一页纸概要**（pitch sheet）：
  - 剧名+类型+集数+一句话梗概
  - 核心卖点（3个）
  - 目标受众
  - 对标作品
- **完整大纲**：outline.md
- **人物简介**：character.md精简版
- **样集剧本**：EP-01 + EP-20（展示开场和付费点）

### 3.3 定价参考
| 类型 | 价格范围 | 条件 |
|------|---------|------|
| 新人编剧 | 2-5万/部 | 无代表作 |
| 有经验编剧 | 5-15万/部 | 有上线作品 |
| 爆款编剧 | 15-50万/部 | 有爆款经历 |
| IP+剧本 | 按收益分成 | 保留IP，分成5-15% |
