# 拍摄制作（第4-12周）

## 一、拍摄排期原则

### 1.1 科幻短剧特殊性
- 场景相对集中（基地、飞船、悬崖反复出现）
- 按场景集中拍摄，而非按集数顺序
- 一天可拍3-5集（短剧每集1-3分钟）
- 总拍摄周期：15-25天

### 1.2 分组拍摄建议

| 拍摄批次 | 天数 | 内容 | 场景 |
|---------|------|------|------|
| 第1批 | 5天 | EP1-20（基地内戏） | 基地室内 |
| 第2批 | 2天 | EP1/20/80等悬崖戏 | 外景悬崖 |
| 第3批 | 3天 | EP21-35（洞穴戏） | 洞穴/地下 |
| 第4批 | 5天 | EP41-60（飞船戏） | 飞船内部 |
| 第5批 | 3天 | EP61-80（太空戏） | 绿幕棚 |
| 第6批 | 2天 | EP81-100（高维空间） | 特效棚/绿幕 |

### 1.3 苏雪晴双版本拍摄
- 苍老版（主线）：先拍，化妆时间长
- 年轻版（闪回EP83等）：后拍，减少化妆切换
- 如用同一演员：每天只安排一个版本

## 二、拍摄规范

### 2.1 短剧拍摄技巧
- **竖屏9:16**（当前主流） or **横屏16:9**（精品路线）
- **特写为王**：80%镜头给脸部特写，情绪直给
- **对话镜头**：正反打为主，节奏快
- **运动镜头**：追跑/冲突场景用手持跟拍
- **灯光**：科幻用冷色调蓝光，情感戏用暖光

### 2.2 每日拍摄流程
```
07:00  化妆+准备
09:00  第一场开拍
12:00  午餐
13:00  第二场开拍
17:00  第三场开拍
19:00  收工+复盘
```

### 2.3 现场必备
- 场记板/场记表
- 台词提词器（手机APP即可）
- 备用电池/存储卡
- 收音设备（小蜜蜂无线麦）
- 每场拍摄前对词一遍

## 三、质量控制

### 3.1 每日检查
- 当天素材立即备份（至少2份）
- 导演/编剧当晚看回放
- 标记需要补拍的镜头
- 确认台词与剧本一致

### 3.2 关键场景重点保障
| 集数 | 场景 | 重点 |
|------|------|------|
| EP1 | 悬崖重逢 | 苏雪晴转身的表演+林远航嘶吼 |
| EP20 | 悬崖坠落 | 安全保障+特效配合 |
| EP40 | 星球爆炸升空 | 特效预留素材 |
| EP80 | 苏雪晴化为光 | 表演+灯光+后期特效三合一 |
| EP95 | 改写重逢 | 全剧情感高潮，给足拍摄时间 |

## 四、后期制作（第12-16周）

### 4.1 剪辑要点
- 每集控制在1-3分钟，宁短勿长
- 开头3秒必须抓眼球（不要黑屏淡入）
- 对话节奏快，删掉所有停顿
- 每集结尾卡在悬念点

### 4.2 字幕+音效
- 必须加字幕（短剧观众大量静音观看）
- 关键台词放大加粗
- 配乐：科幻氛围音乐+情感钢琴曲交替
- 音效：飞船/警报/爆炸/心跳等增强沉浸感

### 4.3 AI辅助后期
- **剪映**：自动字幕+基础剪辑
- **Runway**：AI视频特效
- **Midjourney**：背景/概念图生成
- **ElevenLabs/配音秀**：AI旁白配音

### 4.4 交付标准
- 分辨率：1080P以上
- 帧率：30fps
- 格式：MP4 (H.264)
- 每集片头：剧名+集数（2秒）
- 每集片尾：下集预告（5秒）+ 关注引导
