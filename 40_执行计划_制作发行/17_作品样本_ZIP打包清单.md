## 一、三种 ZIP 包用途与差异

| 用途 | 包名 | 内容范围 | 加密 |

cp 00_主创项目_星海23年/episode_index.md /tmp/星海23年_版权登记包/04_集目录.md
cp 00_主创项目_星海23年/rhythm.md /tmp/星海23年_版权登记包/05_节奏控制表.md
cp -r 00_主创项目_星海23年/01_短剧正文_100集 /tmp/星海23年_版权登记包/06_短剧剧本_100集
cp -r 00_主创项目_星海23年/02_小说原版_100章 /tmp/星海23年_版权登记包/07_长篇小说_100章

联系：TowerTrina338@zohomail.com

# 打包（不加密）
cd /tmp && zip -r 星海23年_版权登记包_v1_20260425.zip 星海23年_版权登记包/

# 移到项目目录
mv /tmp/星海23年_版权登记包_v1_20260425.zip /home/yake/AI_Short_Drama/40_执行计划_制作发行/版权证据/

> ⚠️ **注意**：打包后请检查 ZIP 大小是否 ≤ 50MB（中国版权保护中心限制）。如超出，分两个 ZIP 上传（剧本一份 + 小说一份）。

## 三、B 包｜区块链存证包（不加密）

**用途**：上传到蚂蚁链 / 百度图腾，让系统计算 hash。

**内容**：与 A 包完全相同。

**打包命令**：可直接复用 A 包，重命名即可：

```bash
cp 40_执行计划_制作发行/版权证据/星海23年_版权登记包_v1_20260425.zip \
   40_执行计划_制作发行/版权证据/星海23年_存证包_v1_20260425.zip
```

> ⚠️ 蚂蚁链单文件限制通常 ≤ 100MB，如超出可只存证前 50 集 + 全部小说。

## 八、自动化脚本（可选）

如果你希望一次性生成所有 4 个 ZIP，可以请 AI 生成 `package_all.sh`，但**该脚本中绝不能含密码明文**，密码应作为环境变量或运行时参数。

示例（伪代码）：

```bash
PWD_C=$1  # 命令行传入 C 包密码

# 然后执行打包...
```

> 实际生产时建议手动执行单步命令，避免脚本失误。
