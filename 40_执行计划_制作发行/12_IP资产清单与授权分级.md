## 七、授权分级矩阵（核心定价框架）

按"独家/非独家 × 区域 × 时长 × 链路"四维定价：

### 7.1 独家 vs 非独家
| 模式 | 价格倍数 | 说明 |
|---|---|---|
| **完全独家**（全球+全链路+永久） | ×3.0 | 仅一次性卖断，慎用 |
| **链路独家**（如仅短剧独家） | ×1.8 | 推荐主力模式 |
| **区域独家**（如仅大陆/仅海外） | ×1.5 | 适合多区域分销 |
| **非独家** | ×1.0 | 灵活授权 |

### 7.2 区域定价
| 区域 | 系数 |
|---|---|
| 大陆 | 1.0 |
| 港澳台 | 0.3 |
| 东南亚 | 0.6 |
| 北美 | 1.2 |
| 欧洲 | 0.8 |
| 中东 | 0.4 |
| 全球打包 | 2.5 |

### 7.3 时长系数
| 期限 | 系数 |
|---|---|
| 1 年 | 0.4 |
| 3 年 | 0.7 |
| 5 年 | 1.0 |
| 10 年 | 1.4 |
| 永久 | 2.0 |

### 7.4 计价示例

**示例 1**：某制作方要求《星海23年》100 集剧本 + 大陆独家 + 5 年期 + 短剧链路
- 基础价：100 万
- 链路独家系数：×1.8
- 大陆区域：×1.0
- 5 年期：×1.0
- **合计**：**180 万**

**示例 2**：某出版社要求小说全文 + 全球非独家 + 5 年期 + 实体书+电子书链路
- 基础价：15 万
- 非独家：×1.0
- 全球：×2.5
- 5 年：×1.0
- **合计**：**37.5 万**

## 八、签约必备条款清单

每份授权合同必须包含：

- [ ] **授权范围**：哪些链路（短剧/长剧/小说/漫画/动画/游戏/海外/周边）
- [ ] **授权区域**：精确到国家/地区
- [ ] **授权期限**：起止日期
- [ ] **独家性**：完全独家/链路独家/区域独家/非独家
- [ ] **付款方式**：一次性 vs 分期 vs 流水分成 vs 混合
- [ ] **改编权限**：能否大改 / 哪些不可改（如主角结局）
- [ ] **署名权**：编剧署名（不可转让）
- [ ] **续作权**：保留 / 转让
- [ ] **违约金**：通常为合同金额的 30-50%
- [ ] **管辖法院**：建议约定北京 / 杭州互联网法院
