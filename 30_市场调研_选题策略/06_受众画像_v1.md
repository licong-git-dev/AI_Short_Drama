## 一、核心受众三人组

### 👩 画像 1｜情感投射型｜「等待型小红」（占比预估 50%）

| 字段 | 内容 |
|---|---|
| **性别** | 女 |
| **年龄** | 25-35 岁 |
| **地域** | 三四线城市为主 + 一二线"漂"群体 |
| **职业** | 行政文员、护士、教师、电商客服 |
| **收入** | 月入 4,000-8,000 元 |
| **教育** | 大专-本科 |
| **婚恋** | 单身/失恋/异地恋/已婚 |
| **平台习惯** | 抖音 > 红果 > 番茄 > 快手 |
| **付费意愿** | 中高（单部 9.9-29.9 元可接受） |
| **看剧时段** | 通勤路上 + 睡前 + 午休 |

**情感诉求**：
- 渴望被看见、被等待、被一个人无条件爱着
- 对"长情、专一、痴情"角色有强烈代入感
- 享受"虐到极致"的情感宣泄
- 愿意为"真爱终于被看见"的瞬间付费

**这部剧的钩子点**（针对她）：
- ✅ EP-03 苏雪晴 23 年每天打扫"远航的房间"
- ✅ EP-19 8401 天日志逐页翻飞
- ✅ EP-80 苏雪晴第一次说"我爱你"
- ✅ EP-95 改写时间线后的圆满拥抱

**宣发抓手**：
- 短视频切片台词："我每一天都在等你"
- 海报主视觉：满头白发的女人 vs 年轻的男人，反差对照
- 评论区话术："谁懂啊，等了 23 年的女人"

## 三、目标受众反推：哪些人不是我们的目标

### ❌ 不要硬投的群体
| 群体 | 为什么不投 |
|---|---|
| 50+ 中老年（短剧已饱和） | 对科幻概念接受度低，更喜欢家庭伦理 |
| 12-17 岁未成年 | 付费意愿低，平台监管严 |
| 硬核科幻男（B 站科幻硬核圈） | 会以"科学不严谨"为由 diss 本作 |
| 海外纯英语用户（出海前） | 需要本地化翻译 + 文化适配 |

### ⚠️ 谨慎沟通的群体
| 群体 | 注意点 |
|---|---|
| 影评博主（"中国版星际穿越"标签党） | 拒绝此标签，强调原创 |
| 女权 KOL | 强调苏雪晴主动建设新人类的独立性，避免"恋爱脑"标签 |

## 四、用户故事 ×3

### 用户故事 1：等待型小红
> 林女士，30 岁，杭州某医院护士，单身。每天倒夜班，凌晨 4 点下班路上刷抖音。
>
> "刷到一个白发奶奶站在悬崖边的视频，下面写'她说，再等一天，她已经等了 23 年'。我直接哭了。
> 然后我发现这是个短剧，搜到《星海23年》，没想到红果上免费看到 EP-19，又下了番茄付了 18.9 解锁全集。一晚上没睡，看到 EP-80 苏雪晴说'我爱你'化光那段，我又哭了一次。
> 第二天我把链接发给了我闺蜜，她也熬夜看完。"

**关键洞察**：
- 抖音切片 → 平台搜索 → 免费段试看 → 高情绪节点付费
- 闺蜜传播是关键裂变路径
- 付费点：EP-20 后第一集 + EP-80 强情绪点

### 用户故事 2：陪刷型大刘
> 刘先生，38 岁，江苏县城建材店老板，已婚，两个孩子。
>
> "老婆晚上一直刷一个剧，我边吃饭边瞄了几眼。看到男主独闯辐射区为救老婆吞辐射水那段，我就坐下来一起看了。
> 后来看到孙子驾核弹冲黑洞赎罪那集，我哭了。这剧不娘，挺好看的。
> 老婆付费的，我不付。但她付的我陪刷。"

**关键洞察**：
- 男性是"被动观众"但能形成口碑
- 关键钩子是"硬核 + 牺牲 + 男性责任感"
- 不直接付费，但影响家庭付费决策

### 用户故事 3：嗑糖型小芳
> 王同学，22 岁，广州某 985 大三女生。
>
> "刚开始是在小红书看到一篇'CP 高甜剪辑'，把我吸进去了。
> 然后跑到红果免费看，看完发现这剧封神了。我立马在 B 站做了一个 CP 剪辑发布，3 天涨了 8000 粉。
> 现在我们 CP 群里 200 多个人，每天都在'嗷嗷叫'。我还写了一篇番外发在 Lofter 上。"

**关键洞察**：
- 二创裂变力极强（小红书种草 → B 站剪辑 → Lofter 同人）
- 不直接付费，但带动百倍流量
- 应主动给二创提供原素材包

## 五、调研字段模板（v2 用真实数据校准）

> 用于后续在抖音/小红书评论区抓取数据，或自建用户问卷

```yaml
基础字段:
  - 性别
  - 年龄段（< 18 / 18-24 / 25-34 / 35-44 / 45-54 / 55+）
  - 地域（一线/新一线/二线/三线/四线/县城/农村）
  - 职业大类
  - 收入区间
  - 婚恋状态

行为字段:
  - 主要短剧平台（抖音/红果/番茄/快手/微信视频号/B 站/小红书）
  - 月均看剧时长
  - 月均付费金额
  - 单部最高付费金额
  - 看剧时段
  - 单次看剧时长

偏好字段:
  - 偏好题材（多选）
  - 偏好的钩子类型（虐/爽/反转/CP/亲情/复仇）
  - 厌恶的元素（多选）
  - 是否参与二创/同人

针对本剧:
  - 是从哪里听说的？
  - 第一次刷到的画面/台词
  - 看到第几集决定付费的？
  - 最戳的 3 个瞬间
  - 最想吐槽的 3 个点
  - 是否会推荐给朋友？为什么？
  - 是否会看续作/衍生作？
```
