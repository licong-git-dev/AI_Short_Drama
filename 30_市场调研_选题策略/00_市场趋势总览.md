# 短剧市场趋势总览（2025-2026调研）

> 调研时间：2026年3月
> 数据来源：36氪、虎嗅、澎湃新闻、新榜、知乎等权威媒体

---

## 一、市场规模

| 指标 | 数据 |
|------|------|
| 2025年市场规模 | 677.9亿元（距千亿仅一步之遥） |
| 用户规模 | 6.96亿（超半数中国网民） |
| 红果短剧月活 | 2.12亿，同比增长178.99% |
| 精品短剧占比 | 从2024年15% → 2025年42% |
| AI漫剧抖音全年播放量 | 757.72亿次 |

## 二、商业模式大变局

- **付费模式萎缩**，免费+广告成主流（免费观看占比66.3%）
- 广告收入预计增长3倍，成行业第一大现金流
- "8天充值破亿"神话已成历史
- 头部剧仍可达500万美元+海外收入

## 三、制作成本升级

| 级别 | 成本 |
|------|------|
| 普通剧 | 40-70万/部 |
| S级精品 | 150-300万/部 |
| 早期对比 | 仅1-2万/部 |

## 四、行业演进阶段

```
1.0 粗糙版快餐（2022-2023）
    ↓
2.0 精致版快餐（2024）
    ↓
3.0 精致版正餐（2025-今）
```

观众需求已从"爽就完了" → "要有逻辑、有细节、有情怀"

---

## 五、2026年六大预测

1. **IP系列化开发**：单点爆款 → 长青系列（《云渺系列》《太奶奶系列》）
2. **AI漫剧精品化**：新中式奇幻、互动漫剧、女频成长题材爆发
3. **女频漫剧崛起**：古风/甜宠/成长类即将爆发（复制男频先行路径）
4. **IP资产全模态化**：短剧+AI游戏+数字周边同步开发
5. **垂直行业大模型**：专针对短剧叙事的行业模型取代通用AI工具
6. **出海竞争深化**：从走量 → 服务体系化+价值深度化

---

## 六、监管红线（必须遵守）

- 禁止"怼""打脸"等泄愤式表达
- 禁止极端人物设定
- "穿越""重生"需符合生活与历史逻辑
- 爱情题材须倡导正确婚恋观
- AIGC内容须特别标注，专项治理至2026年3月
