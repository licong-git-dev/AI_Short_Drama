# 创作行动清单（基于市场调研）

> 状态标记：⬜ 待做 / 🔄 进行中 / ✅ 完成

---

## 第一阶段：选题确定（优先级最高）

- [ ] ⬜ **选定核心题材方向**（参考01_题材分析）
  - 候选A：地域文化×现实主义（东北/川渝/广东）
  - 候选B：现实女频新视角（独立女性成长）
  - 候选C：AI漫剧×新中式奇幻
  - **决策依据**：差异化程度 + 制作可行性 + 社会情绪契合

- [ ] ⬜ **确定集数与商业模式**
  - 免费模式（红果，100集+，靠播放量）
  - 付费模式（抖音/快手，80-100集，设付费关卡）
  - 推荐：80-100集，第20集设付费转化点

- [ ] ⬜ **锁定核心爽点**（一句话描述核心爽感）
  - 示例："东北老厂工人下岗后白手创业，20年后让曾经欺压他的领导跪地道歉"
  - 参考市场：精准社会情绪 + 强共鸣叙事

---

## 第二阶段：剧本架构

- [ ] ⬜ **写故事大纲**（`/outline`指令）
  - 必须包含：核心冲突 / 主要转折 / 结局类型
  - 参考法则：短剧创作法则 + 三级悬念矩阵

- [ ] ⬜ **设计人物标签**（`/character`指令）
  - 第1集必须建立主角核心标签
  - 反派动机清晰，不模糊化
  - 配角功能明确，每个配角有且只有一个核心作用

- [ ] ⬜ **规划爽点节奏表**（`/catalog`指令）
  - 按集数标注爽点类型（小高潮/反转/大爆点）
  - 第20-21集付费转化节点设计
  - 确保虐点≤3集必释放

---

## 第三阶段：剧本创作

- [ ] ⬜ **写第1-10集**（`/write 1` ~ `/write 10`）
  - 重点：前3集强开场，第1集必须有小爽点
  - 检查：是否每集有3个钩子

- [ ] ⬜ **写第11-20集**
  - 重点：免费段末尾强悬念积累
  - 第20集结尾：强冲突/大反转

- [ ] ⬜ **写第21集+**
  - 付费后立即给情感回报
  - 每5集中型爽点保证节奏

---

## 第四阶段：质量提升清单

每集完成后检查：

### 内容质量
- [ ] 前3秒有视觉冲击/信息密度
- [ ] 每集3个强钩子（开场/中段/结尾）
- [ ] 对话：无填充词，第一句有实质信息
- [ ] 动作描写：极简，无心理渲染
- [ ] 结尾：留钩子（未完成动作/说一半的话/突然出现的人）

### 精品化要求（2025新标准）
- [ ] 人物行为有逻辑动机
- [ ] 至少1个烟火气细节增加真实感
- [ ] 至少1个可讨论的"话题点"
- [ ] 情感深度不只靠爽感，有情怀表达

### 合规检查
- [ ] 无"怼""打脸"泄愤式语言
- [ ] 无不符合逻辑的极端人物设定
- [ ] 穿越/重生如有，符合历史逻辑
- [ ] 爱情关系呈现正确婚恋观

---

## 第五阶段：宣发准备

- [ ] ⬜ **整理每集"话题点"**（共计10个以上）
- [ ] ⬜ **准备热搜话题词**（每10集3个）
- [ ] ⬜ **设计评论区话题**（引导"你们猜结局"讨论）
- [ ] ⬜ **准备地域联动**（如果有地域题材，联系文旅）
