## 一、平台分账规则

### 抖音辰星计划
| 层级 | 分成比例 | 触发条件 |
|------|----------|----------|
| 基础分成 | 覆盖制作成本 | 默认 |
| 流量激励 | +奖励 | 播放量+完播率达标 |
| 品类奖励 | 额外10%流量分成 | 稀缺题材（悬疑/现实） |
| 综合最高 | 25% | 符合精品标准 |

**稀缺题材优先**：悬疑、现实题材有额外奖励→选题时可优先考虑

### 快手星芒计划
| 触发条件 | 分成比例 |
|----------|----------|
| 默认基础 | 12% |
| 单集有效播放>500万次 | 18% |
| 带动用户付费转化 | 叠加+20% |

**快手特点**：下沉市场用户占62%，生活类短剧互动率是都市题材2.3倍

### 红果短剧
- 免费模式为主（无付费墙）
- 靠广告分成+品牌合作
- 月活2.12亿，增速178.99%
- 适合：现实主义/生活类/方言地域题材

## 三、付费关卡设计技巧

### 标准付费漏斗
```
免费集（1-20集）→ 强悬念截止 → 付费解锁
                    ↑
               付费转化关键点
```

### 最优付费节点设置
```
第18-20集：给予小高潮（预热情绪）
第20集结尾：悬念/反转/冲突极点
第21集开头：付费后立即给情感回报
→ 用户体验：付了钱马上值回去
```

### 付费意愿提升技巧
1. **悬念积累法**：在免费段埋下多个未解答的问题
2. **角色情感投资**：让观众对角色产生强烈情感依附
3. **暗示法**：第20集台词暗示"真正的秘密即将揭晓"
4. **话题讨论法**：在评论区制造"你们猜结局是什么"热度

## 四、多元变现路径（2025趋势）

| 变现方式 | 特点 | 适合场景 |
|----------|------|----------|
| 广告分成 | 免费模式主流 | 红果/高播放量 |
| 单集付费 | 转化率最高 | 悬疑/情感沉浸类 |
| 品牌定制 | 整合植入 | 有IP号召力 |
| 短剧+文旅 | 带动旅游消费 | 地域题材 |
| IP系列化 | 长效变现 | 建立粉丝基础后 |
| 出海 | 东南亚高转化 | 普世情感题材 |

> 示例：某现实题材带动当地旅游收入超5亿元

## 五、宣发策略（2025新要求）

爆款不只靠内容好，宣发同等重要：

1. **站外热搜**：抖音头条/微博头条炒热搜
2. **主创路演**：多城市路演，直接触达用户
3. **话题制造**：每10集至少3个可讨论话题点
4. **口碑发酵**：豆瓣/知乎种草，长尾效应
