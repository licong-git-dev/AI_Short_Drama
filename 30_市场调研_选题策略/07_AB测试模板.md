## 一、测试方法论（统一标准）

### 1.1 三大基本原则
1. **单变量原则**：一次只改一个变量（封面 / 文案 / 时长 / 节点）
2. **样本量原则**：每组至少 1,000 次展示才算有效，统计显著性 p < 0.05
3. **成本止损原则**：单条 A/B 测试预算 ≤ 1,000 元，未达显著性立即砍

### 1.2 数据指标分级

| 指标级别 | 指标名 | 用于判断 |
|---|---|---|
| **L1 表层** | CTR（点击率） | 封面/标题吸引力 |
| **L2 中层** | 完播率 / 平均观看时长 | 内容钩子强度 |
| **L3 深层** | 付费转化率 / 加书架率 | 付费意愿 |
| **L4 底层** | 留存（次日/7 日）/ 复访 | 长期价值 |

### 1.3 测试结果判定

| 差异幅度 | 判定 | 行动 |
|---|---|---|
| > 30% | 显著优胜 | A 版全量上线 |
| 10-30% | 优势明显 | A 版主推，B 版辅助 |
| 5-10% | 弱优势 | 加大样本继续测，或并行投放 |
| < 5% | 无显著差异 | 选制作成本更低的版本 |

## 四、A/B 测试场景 3｜付费节点位置

### 测试目标
找出付费转化率最高的付费节点

### 候选方案

| # | 付费节点 | 截止剧情 | 推断逻辑 |
|---|---|---|---|
| **C1** | **EP-18 截止** | 苏雪晴吐血倒地后立即付费 | 利用最强悬念瞬间 |
| **C2** | **EP-20 截止**（标准） | 8401 天日志 + 悬崖告白 + 地震坠落 | 完整情绪高潮 |
| **C3** | **EP-22 截止** | 地底洞穴 + 苏雪晴"你终于抱我了" | 给一点情感满足后付费 |

### 三种节点的预期表现

| 节点 | 优势 | 劣势 |
|---|---|---|
| **C1（EP-18）** | 悬念最强 → 转化率高 | 用户感觉"投资不够" → 退订率高 |
| **C2（EP-20）** | 行业最佳实践 → 心理预期一致 | 同质化 |
| **C3（EP-22）** | 情绪释放后理性付费 → 留存率高 | 已经"被满足"→ 转化率低 |

## 八、A/B 测试通用 Checklist

每次测试前确认：

- [ ] 单变量？（其他条件全部一致）
- [ ] 样本量足够？（每组 ≥ 1,000）
- [ ] 预算止损线设定？（CPC ≥ X 立即停）
- [ ] 数据收集表填写时间点确定？（每 4 小时一次）
- [ ] 决策标准提前写明？（差异 > X% 才采纳）
- [ ] 复盘文档模板准备好了？（W1 复盘 → W2 复盘）

每次测试后产出：

- [ ] 数据收集表完整填写
- [ ] 优胜版本明确
- [ ] 失败原因分析（即使输的也要记录原因）
- [ ] 下轮测试假设
