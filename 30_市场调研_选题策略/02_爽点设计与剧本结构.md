# 爽点设计与剧本结构标准

---

## 一、工业化剧本结构标准

### 黄金3秒法则
```
第1-3秒：视觉冲击 + 高信息密度
→ 目标：让观众停止划走
→ 手段：强冲突画面/悬念台词/极端场景
→ 禁止：平淡开场、环境渲染、人物介绍
```

### 每集3个强钩子
```
钩子1（开场）：制造强悬念/冲突
钩子2（中段）：反转或升级
钩子3（结尾）：留下更大悬念→推动下集点击
```

### 三级悬念矩阵
```
表层悬念（每集）：这集结局怎样？
中层冲突（每5-10集）：这段关系/对抗走向何方？
深层阴谋（全剧）：最终真相/身份揭秘
```
> 某探案类短剧用三重悬念设计，完播率提升至41.7%

## 二、爽点分布节奏表

| 集数段 | 爽点类型 | 功能 |
|--------|----------|------|
| 第1集 | 强冲突开场钩子 | 3分钟内必须抓住观众 |
| 第1-10集 | 每2-3集小高潮 | 建立观看习惯 |
| 第11-20集 | 免费末段强悬念 | 激发付费欲望 |
| 第20-21集 | 重大转折/悬念 | 付费转化黄金节点 |
| 第21集后 | 每5集中型爽点 | 维持付费用户 |
| 每20集 | 大爆点 | 制造社交传播话题 |
| 全剧 | 至少10个反转 | 保持新鲜感 |

---

## 三、开场设计必检清单（前3集）

- [ ] 强冲突开场：被欺辱/身份曝光/意外事件
- [ ] 主角困境展示：让观众产生共情
- [ ] 悬念设置：引发好奇心，不能当集解决
- [ ] 第一个小爽点：即时情感回报（不能让观众白等）
- [ ] 第1集建立主角核心人设标签

## 四、对话写作规范

### 正确示范
```
陈薇推开门：这是我家，你凭什么坐着？
张总裁站起来：（撕合同）我买下来了。
```

### 错误示范（禁止）
```
陈薇内心无比愤怒，她深吸一口气，
颤抖着嘴唇说：你...你为什么这样对我？
张总裁慢慢站起来，嘴角微微上扬...
```

### 对话黄金法则
- 第一句话必须有实质信息
- 删掉所有"你知道吗""其实吧""所以说"
- 每句台词≤15字
- 用动作替代心理描写

## 五、动作描写规范

| 写法 | 示例 |
|------|------|
| ✅ 极简直接 | 他扇了她一巴掌 |
| ✅ 动作串联爽点 | 推门—亮身份—全场跪 |
| ❌ 过度渲染 | 他愤怒地扬起右手，狠狠地... |
| ❌ 心理铺垫 | 她感觉心里有什么东西碎了... |

## 六、情感曲线设计

```
甜宠剧：甜虐比 = 7:3
复仇剧：前3-5集虐，后期持续爽
虐点规则：压抑不超过3集，必须释放
节奏规律：虐点→爽点→虐点（交替避免情绪疲劳）
```

### 付费转化情感设计
```
第18-20集：小高潮（给付费前的情感回报预告）
第20-21集：强悬念/重大转折（激发付费）
付费后立即：情感大爆发（让付费用户值回票价）
```

## 七、2025年新增"精品化"要求

从2025年起，观众已不满足纯爽感，需增加：

1. **内在逻辑性**：人物行为有动机，不能强行反转
2. **情感细节**：1-2个烟火气细节增加真实感
3. **价值表达**：1个可讨论的"话题点"（社交传播关键）
4. **结局处理**：不强行大团圆，留余味比强行完美更好
